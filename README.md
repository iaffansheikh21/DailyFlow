# Todo App

A full-stack Todo application built with **Next.js**, **MongoDB**, and **Node.js**. Users can register, login, create, edit, delete, and filter todos. Includes authentication features like **forgot/reset password**.

---

## Tech Stack

- **Frontend:** Next.js, React, Tailwind CSS (or your styling library)  
- **Backend:** Node.js, Next.js API Routes  
- **Database:** MongoDB  
- **Authentication:** JWT, Password Hashing (bcrypt)  
- **Email Service:** NodeMailer / any SMTP service (for reset password)  
- **Deployment:** Vercel / any Node.js supported server  

---

## Features

- User registration and login  
- Forgot password and reset password  
- Create, edit, delete todos  
- Filter and search todos by priority or status  
- Proper error handling and validation
